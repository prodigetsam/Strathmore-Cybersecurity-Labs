# Data Network Design and Management Assignment

This project focused on building and configuring a scalable network in Cisco Packet Tracer using both IPv4 and IPv6 addressing. 
The network was designed to support different departments with different host requirements, including one very large subnet.

# Project Overview

The network included:

2 Routers
4 Switches
8 End Devices
# Host Requirements
Subnet	Number of Hosts
Subnet 1	10,000
Subnet 2	147
Subnet 3	32
Subnet 4	7
# What I Implemented
- Used VLSM subnetting with a Class A private IPv4 range
- Configured IPv4 and IPv6 addressing
- Assigned Global Unicast and Link-Local IPv6 addresses
- Configured routers and switches using the required naming format
- Tested connectivity between networks using ping
